# 🍔 Sabor na Brasa

Projeto de uma hamburgueria desenvolvido utilizando **HTML**, **CSS** e **JavaScript**, com foco em um visual moderno, responsivo e intuitivo.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## ✨ Funcionalidades

- Cardápio digital
- Layout responsivo
- Navegação intuitiva
- Sessão de promoções
- Botão para WhatsApp
- Animações e interatividade
- Interface moderna

## 🎯 Objetivo do Projeto

O objetivo deste projeto é criar uma experiência digital moderna para a hamburgueria **Sabor na Brasa**, facilitando a apresentação dos produtos e aproximando os clientes da marca.

---

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em:

- Celulares
- Tablets
- Notebooks
- Desktops
